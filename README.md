math-label-c-
=============

How to draw a expression:<br />
Use -sqrt- to start a square root and -esqrt- to end it (Example: -sqrt- 64 -esqrt- = 8)
Use -n- to go in a new line.
Use nominator/denominator to create a fraction

Full examples:
1. -sqrt- 4/64 -esqrt- = 2/8
2.
-sqrt- (8 x 8) -esqrt- -n-
-sqrt- 64 -esqrt- = 8
